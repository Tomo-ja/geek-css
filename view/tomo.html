Line height

The line-height property defines the amount of space above and below.
This is work for inline elements so a element is set to display: inline or inline-block.

This property accept the keyword values "normal" or "none" as well as number, length, percentage.
if number set as values without unit, then total height of line will be the number of value times the height of text itself.

Letter spacing

The letter-spacing property defines the amount of space between each letter in a given element or block of text.

This property accept font-relative values(em, rem), percentage(%), absolute values(px) and the keyword: normal.

Most important point to use letter-spacing is every value you set ADD to the default spacing based on the font. On the other hand, you will set negative values if you want to make the spaces less tighten than the default.

Text decoration

The text-decoration property adds an underline, overline, line-through, or a combination of lines on selected text.

This property accept: none, underline, line-through, overline, and inherit.

By default, the line is set 1px, solid, and the same color of the text.

However, it can be edited by other properties:

 text-decoration-color: set color of a line.
 text-decoration-style: set style of a line, like wavy, double dashed, or dotted.
 text-decoration-thickness: set bold of a line.
 text-underline-offset: set a distance between text and a line.

Text shadow

The text-shadow property will set a shadow behind of a text. You can set an area respectively and color of shadow. Plus, you can apply multiple text shadow.

This property accept 4 values at the same time.

 First value: the horizontal distance of shadow
 Second value: the vertical distance of shadow
 Third value: the blur radius of shadow
 Fourth value: the color of the shadow.

Using positive numbers as the first values and second values place a shadow to the right and blow of the text respectively
In contrast, using negative numbers place a shadow to the left and up.

The third value, the blur radius, is an optional value. It's the amount of pixels the text is stretched which causes a blur effect. If you don't set this value, it's automatically set as zero.
